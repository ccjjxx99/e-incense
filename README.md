# e-incense
一个环保又庄重的 **电子烧香网页**，让大家在云端供奉神明、寄托心愿。  
不再需要纸钱和香火，减少污染，同时保持敬意。

## ✨ 特性
- 🕯️ **电子上香**：点击即可点香，动画效果模拟真实的香火。
- 🙏 **许愿祈福**：输入心愿，存放在云端（可选持久化）。
- 🌏 **环保理念**：减少纸张与烟雾污染，用科技表达虔诚。
- 🎨 **界面简洁**：仿庙宇风格 UI，宁静氛围。

## 🛠️ 技术栈
- **前端**：Vue
- **动画**：CSS

## 🚀 快速开始
```bash
# 克隆仓库
git clone https://github.com/ccjjxx99/e-incense.git
cd e-incense

# 安装依赖
npm install

# 启动开发环境
npm run dev
